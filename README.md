# Gemini Premium Store

## Files
- `index.html`
- `style.css`
- `script.js`
- `assets/`

## What is included
- Premium animated hero slider using your uploaded lifestyle photos
- Product catalog built from your coffee pack images
- Search, filters, and sorting
- Wishlist
- Shopping cart
- Quantity controls
- Checkout modal
- Bundle add button
- Persistent cart and wishlist via localStorage
- Responsive design for desktop and mobile

## Run in VSCode
1. Open the folder in VSCode
2. Open `index.html`
3. Run with Live Server for the best experience

## Notes
- This is a frontend-only store
- Cart, wishlist, and checkout are simulated in browser
- You can later connect it to a real backend, payment system, or CMS
